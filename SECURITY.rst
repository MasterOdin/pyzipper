Security
========

If you have discovered a security vulnerability in this project, please report it
privately. **Do not disclose it as a public issue.** This gives us time to work with you
to fix the issue before public exposure.

Please submit the report as a `Github draft security advisory`_. Please include
a description of the impact and how to reproduce it.

.. _`Github draft security advisory`: https://github.com/danifus/pyzipper/security/advisories
